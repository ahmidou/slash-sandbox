Detailed Description

template<typename T>
struct nih::Histogram< T >

A small helper class to compute histograms. Usage of this class is supposed to be coupled with Stats<T>. In order to use this class, one would typically loop once over the values to compute basic statistics with a Stats<T> object, and loop over the values once again to create a Histogram<T>.
